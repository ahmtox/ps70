Project Proposal

The assignment for this week to code an Arduino to do something, so I knew right off the bat I wanted to make something work with LED's. I learned how to light up an LED in arduino so I thought it might be cool to extend this using two LEDs instead of just one. I didn't know how to exactly wire things together, so I had to watch a few tutorials on wiring components in Arduino.

Of the materials that I used, I only used Breadboard, some wires, my ESP32 equivalent, 2 LED's, and 2 100 ohm resistors.

The code below is what I used in conjucation with my ESP32 XIAO C3

                
    int yellow = D0;
    int red = D1;

    void setup() {
        pinMode(yellow, OUTPUT);
        pinMode(red, OUTPUT);
    }

    void loop() {
        digitalWrite(yellow, HIGH);
        digitalWrite(red, HIGH);
        delay(1000);                   
        digitalWrite(yellow, LOW); 
        digitalWrite(red, LOW);
        delay(1000);              
    }
